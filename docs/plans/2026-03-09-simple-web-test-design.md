## 功能设计

### API 端点

1. `GET /` - 首页
2. `GET /api/items` - 获取所有项目
3. `POST /api/items` - 创建项目
4. `GET /api/items/<id>` - 获取单个项目
5. `PUT /api/items/<id>` - 更新项目
6. `DELETE /api/items/<id>` - 删除项目

### 数据模型

```python
{
```
