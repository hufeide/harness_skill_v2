## 新增功能

### 1. dev_complete.sh 脚本

**功能**:
- 🎉 显示开发完成提示
- 📊 自动检测项目类型
- 📝 显示测试命令
- 🎯 提醒测试覆盖率目标
- 📋 列出后续操作步骤
- ⚠️  检查未提交的更改

**使用方法**:
```bash
bash scripts/dev_complete.sh
```

**输出示例**:
```
╔════════════════════════════════════════════════════════╗
║                                                        ║
║     🎉 功能开发完成！                                  ║
║                                                        ║
╚════════════════════════════════════════════════════════╝

✓ 状态已更新：testing - 等待测试验证

════════════════════════════════════════════════════════
  下一步：测试验证
════════════════════════════════════════════════════════

📦 Python 项目 - 运行测试：

  # 运行所有测试
  pytest tests/ -v

  # 带覆盖率报告
  pytest tests/ --cov=src --cov-report=html

════════════════════════════════════════════════════════
  测试覆盖率目标
════════════════════════════════════════════════════════

  ✓ 总体覆盖率：> 80%
  ✓ 核心功能：  > 90%
  ✓ 关键路径：  100%
```

touch docs/plans/$(date +%Y-%m-%d)-user-auth-design.md
# 编辑设计文档...

# 3. 更新状态
bash scripts/update_state.sh update planning "完成需求分析"

# 4. 创建计划
/writing-plans

# 5. 更新状态
bash scripts/update_state.sh update tasks "任务规划完成"

# 6. 开发功能
# 创建文件到正确位置
# src/auth.py
# tests/test_auth.py

# 7. 定期更新状态
bash scripts/update_state.sh update development "实现用户认证"

# 8. 开发完成
bash scripts/dev_complete.sh

# 9. 运行测试
pytest tests/ -v --cov=src

# 10. 更新状态
bash scripts/update_state.sh update verify "测试通过"

# 11. 代码审查
/requesting-code-review

# 12. 完成验证
/verification-before-completion

# 13. 提交代码
git add .
